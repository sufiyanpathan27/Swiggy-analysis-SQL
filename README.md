### Overview

This project analyzes Swiggy order data using SQL. 
The dataset includes customers, orders, restaurants, and delivery partners.

We cleaned the data and performed analysis using SQL queries to understand orders,
revenue, customer behavior, and delivery performance.

Key Insights:

- Total orders are 15000 which shows strong platform activity
- Total revenue generated is 24465659 which indicates good business performance
- Some customers order more frequently than others, showing loyal users
- A few restaurants receive more orders and generate higher revenue
- Most orders are successful compared to cancelled ones
- There are peak hours when orders are highest (mainly lunch and dinner time)
- Some delivery partners handle more orders than others
- Average delivery time is consistent, showing stable delivery performance
- Certain days have higher order volume compared to others
- Order amount varies per customer, but overall spending pattern is stable

Conclusion:

This analysis helps understand customer behavior, restaurant performance, and delivery efficiency. 
It can be useful for improving service, increasing revenue, and making better business decisions.
